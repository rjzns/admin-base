# KASTOR IAM
Внутренняя система управления учётными записями, доступами и ролями для всех информационных систем Группы компаний «КАСТОР»

## Обзор
Система предназначена для внутреннего использования в Группе компаний «КАСТОР» и охватывает все дочерние предприятия холдинга. Она будет интегрирована с существующими базами данных 1С и кадровыми сервисами.

### Основные функции приложения
- Быстрое добавление и блокировка пользователей.
- Интеграцию с Active Directory и кадровыми системами.
- Централизованный контроль доступов.
- В перспективе — инвентаризацию оборудования.
